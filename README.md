My project is about generating a pptx document from GEDCOM Data ( tested with My Heritage )

Export GEDCOM file form your Application
GEDCOM Data are imported in SQLite.
PPTX Document is generation using data saved in SQLite :
Personal biographies (Notes, Occupations, Educations, all events)
Parents, Siblings, Uncles, in Laws
Multi level Descendants trees
Multi Level Ascendants trees
